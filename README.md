## Installation :hammer:

> Install virtual env :
```
$ pip install pipenv
```

> Set env version :
```
$ pipenv --python 3.10
```

> Activate env :
```
$ pipenv shell 
```

> Install dependencies :
```
$ pipenv install -r requirements.txt
```
